    PropaneDB

    V-0.0.1 ·

    Importing

    unlike most lua librarys for propaneDB you need to run the result of require("/wolfos.libs.propaneDB") with a argument

    local propane = require("/wolfos.libs.propaneDB")("db")
    arg result
    ver version
    db the main propaneDB library
